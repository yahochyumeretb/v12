#### Окружение:
- nodejs 18;
- VSCode;
- ESLint плагин для VSCode;
- для отладки необходимы бразузеры Chrome или Edge.

#### Создание пустого проекта:

```commandline
npm create vite@latest ./ -- --template react 
```

#### Установка зависимостей:

```commandline
npm install
```

#### Запуск проекта:

```commandline
npm run dev
```

#### Сборка проекта:

```commandline
npm run build
```
